# stait.ai website starter (Next.js + Tailwind + Motion)

This is a custom, animation-forward marketing site starter for **stait.ai**.

## Local dev

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Deploy (low-cost options)

- **Vercel** (easiest for Next.js): import the repo, deploy, then point `stait.ai` DNS to Vercel.
- **Cloudflare Pages**: works well for static/edge-friendly sites; choose the Next.js preset.
- **Netlify**: also fine; pick the Next.js runtime.

## Customizing

- Copy updates are in `components/sections/*`.
- Brand colors live in `app/globals.css` (CSS vars).

## Domain (Squarespace)

Keep the domain registered at Squarespace and just update DNS records to your hosting provider.
